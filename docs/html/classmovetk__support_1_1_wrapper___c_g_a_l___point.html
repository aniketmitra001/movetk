Detailed Description

template<class Kernel>
class movetk_support::Wrapper_CGAL_Point< Kernel >

A wrapper class for CGAL Point.

This class acts as a wrapper to a CGAL point by encapsulating CGAL specific details of a point. Provides a generic interface for a Point.

Template Parameters
Kernel- A traits class that defines CGAL geometry types
