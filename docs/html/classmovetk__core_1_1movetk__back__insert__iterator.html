Detailed Description

template<class Container>
class movetk_core::movetk_back_insert_iterator< Container >

an adaptor to std::iterator for for inserting at the end of a container

unlike the std::back_inserter, ths iterator has a value_type. Increment operator has no effect. The assignment operator extend the size of the container by inserting an element at the end and returns an iterator to the end
