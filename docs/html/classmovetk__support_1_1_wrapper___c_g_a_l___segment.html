Detailed Description

template<class Kernel>
class movetk_support::Wrapper_CGAL_Segment< Kernel >

A wrapper class for CGAL Segment.

This class acts as a wrapper to a CGAL segment by encapsulating CGAL specific details of segment. Provides a generic inteface for a segment.
