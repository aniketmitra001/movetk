template<class Kernel>
class movetk_support::Wrapper_CGAL_Vector< Kernel >

A wrapper class for CGAL vector.

This class acts as a wrapper to a CGAL vector by encapsulating CGAL specific details of vector. Provides a generic inteface for a vector.
