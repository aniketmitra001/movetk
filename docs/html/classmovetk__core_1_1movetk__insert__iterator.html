Detailed Description

template<class Container>
class movetk_core::movetk_insert_iterator< Container >

an adaptor to std::iterator for inserting into a container at a given position

unlike the std::inserter, ths iterator has a value_type. Increment operator has no effect. The assignment operator extends the size of the container by inserting an element at a given position
